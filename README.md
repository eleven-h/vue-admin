# 硅谷电商后台管理系统

项目描述：基于vue3开发的电商后台管理系统，完成多角色权限控制，数据可视化，商品订单管理，以及用户信息管理，角色权限分配，数据大屏等功能

技术栈：vue3+vite+typescript+pinia+axios+Element-Plus+Echarts
